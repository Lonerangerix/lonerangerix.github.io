Run Guys

Run Guys is a asymmetrical party game where one person plays in VR and tries to eliminate the running players who join via android phones.


Run Guys is a fun project where we made a big VR character, whose goal is to stop the small running characters (they are his toys but can move lol) from reaching the finish line. People can choose to either play as the VR character or the running characters, which is connected using Android phones. The running characters can perform moving left and right, and jumping by the utilization from their phones' accelerometer and gyroscope.

The project's main difficulty was to combine three devices: the VR headset, a computer which is the host, and multiple android phones into one game scene. I was responsible for making this happen, using mirror networking in Unity. It felt awesome the moment everything worked as intended and running together. We had so much fun testing the game together in our team as well. I learned a lot about multiplayer function in Unity from this project. I also made the customization function on Android phones, which was cute.
