B-Bender Guitar in VR

The project succeeded in simulating B-Bender Guitar's bending function in VR. The project mapped the virtual guitar to a real world guitar to achieve this.


This project actually contains a lot of different methods to make it work. It was nice working with my awesome teammates, each of whom having different abilities, from web development, music composing, to MIDI implementation. We were able to send data from an actual MIDI guitar to a sever and read data from that using websockets in Unity and display the visualization in VR headset, mapped to the virtual guitar. And implemented the bending function by attaching the VR controller to the top of the guitar, which would calculated its own translation when the guiar bends, and send data to the server, and play the bending sound in ableton live.

I didn't have much knowledge about music and was amazed how all this communication between different platforms would actually work. I contributed in modeling the virtual guitar, the virtual environment and VR development as well.
