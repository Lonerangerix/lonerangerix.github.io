Spongebob Game

Available on itch.io

I've tried replicating the town from the cartoon SpongeBob into games and added multiple game mechanicss to it. The project took a time-span of over 3 months.


I was both learning and developing this project at the same time. It was also a main highlight project that made my channel grow from 10 thousand subscribers to 100 thousand. The main idea of starting this series was that I wanted the audience to know about how developing a game would look like, what procedures are involved, how the modeling and coding was done in a fun way. I was still learning a lot when developing this, so the acutal game's optimization was a bit off. But I was happy that I managed to implement burger making mechanics through a lot of coding. I was making this as a devlog and posting videos as I was developing it. I've learned lightmap baking and shaders during this project.

You could also catch jellyfish, drive cars, play roller coaster in this game. I've more ideas in mind but this project was gradually growing and I felt like I should wrap it up and learn to make new type of videos and games.
