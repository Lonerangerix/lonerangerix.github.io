Naruto Horror Game

Available on itch.io

With the request from Tencent to help promote their game Naruto Mobile, I made a horror game based on Naruto by myself, suited to their needs.


This is my first horror game. It was a request from Tencent, who sponsored the video, to have me make a horror game based on their game character. So I tried to model some scenes in Naruto, and used first person controler to make the game. Some postprocessing and lightmap baking were done in the game.

I put my hands on timeline component from Unity the first time for this game, because I needed to make cutscenes, and man that was fun. I also didn't have the habbit of using Cinemachine component that much before. But it actually suits really well with the timeline component to make a cutscene. There were some enemy evasion and parkour element in this game.


I personally don't play many horror games but I felt it was fun to make a horror game so that you yourself don't get scared by any of the jumpscares cuz you know exactly when each is going to show up. That was a fun experience.
